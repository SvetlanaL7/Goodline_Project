# Goodline_Project
### Сборка проекта
```swift
  * git clone https://github.com/SvetlanaL7/Goodline_Project.git
  * cd Goodline_Project/Sources/Linux_Goodline_Project
  * swift build
 ``` 
### Запуск проекта
```swift
  swift run Linux_Goodline_Project
```
